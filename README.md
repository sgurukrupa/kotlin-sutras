Four integer types differing by sizes.
-------------------------------------
1. Byte (8)
2. Short (16)
3. Int (32)
4. Long (64)

All are signed types. Following are value ranges:
------------------------------------------------
1. Byte [-128, 127]
2. Short [-32768, 32767]
3. Int [-2e31, 2e31 - 1]
4. Long [-2e63, 2e63 - 1]
